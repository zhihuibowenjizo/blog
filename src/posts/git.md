# git
## git的概念  
Git 是一个版本控制系统，是任何软件开发项目中的主要内容。通常有两个主要用途：代码备份和代码版本控制。你可以逐步处理代码，在需要回滚到备份副本的过程中保存每一步的进度！
## git配置
可以保存 Git 用户名和电子邮件，这样就不必在以后的 Git 命令中再次输入它们。  
在命令行中配置本地仓库的账号和邮箱：
```git
$ git config --global user.name "wupx"  
$ git config --global user.email "wupx@qq.com"  
```
